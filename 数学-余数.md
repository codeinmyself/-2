## 例题
[个位数字为 K 的整数之和](https://leetcode.cn/problems/sum-of-numbers-with-units-digit-k/)
```java
public int minimumNumbers(int num, int k) {
        if (num == 0) {
            return 0;
        }
        for (int i = 1; i <= 10; ++i) {
            if (k * i <= num && (num - k * i) % 10 == 0) {
                return i;
            }
        }
        return -1;
    }
```
