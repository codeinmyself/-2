
## 判断两个数是否是异号
```java
int x = -1, y = 2;
boolean f = ((x ^ y) < 0); // true

int x = 3, y = 2;
boolean f = ((x ^ y) < 0); // false
```
利用的是补码编码的符号位,如果利用乘积或者商来判断两个数是否异号，这种处理方式可能造成溢出。
